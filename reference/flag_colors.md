# Color palette for flagged records

`flag_colors` is a named character vector defining the default colors
used to plot occurrence records flagged with `mapview_here()`.

## Usage

``` r
flag_colors
```

## Format

A named character vector where:

- names:

  Flag labels corresponding to categories generated by the various
  `flag_*` and checking functions.

- values:

  Hex color codes or standard R color names used for plotting.

## See also

`mapview_here`
